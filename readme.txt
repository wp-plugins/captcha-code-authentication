== Description ==
Adds captcha anti-spam methods to the forms for comments, registration, lost password and login. In order to post comments or register, users will have to type in the code shown on the image. This prevents spam from automated bots. Adds security. You can easily configure from Admin panel

== Installation ==

1. Download the plugin.
2. Upload to your blog (/wp-content/plugins/).
3. Activate it.
4. Click the 'WP Captcha' menu.
6. Fill in the options.

Important Note: It is mandatory to save options in this plugin.

You're done!

Uninstalling is as simple as deactivating the plugin.
